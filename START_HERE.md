## 🚀 ЧТО ДЕЛАТЬ ДАЛЬШЕ (ПОШАГОВО):

### 1️⃣ Установи Python (если ещё не установлен)

**Windows:**
- Скачай: https://www.python.org/downloads/
- Запусти установщик
- ⚠️ **ВАЖНО**: Поставь галочку "Add Python to PATH"!
- Нажми "Install Now"

**Проверка:**
Открой командную строку (Win + R → cmd) и напиши:
```bash
python --version
```
Должна показаться версия Python (например: Python 3.11.0)

---

### 2️⃣ Создай Telegram бота

1. Открой Telegram
2. Найди **@BotFather** (официальный бот для создания ботов)
3. Отправь команду: `/newbot`
4. Придумай **имя** (например: "Мой Музыкальный Бот")
5. Придумай **username** (должен заканчиваться на "bot", например: "my_music_bot")
6. **СКОПИРУЙ ТОКЕН!** Он выглядит так: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`

---

### 3️⃣ Настрой проект

1. **Создай файл `.env`:**
   - Скопируй файл `.env.example`
   - Переименуй копию в `.env` (без расширения!)
   - Открой `.env` в блокноте

2. **Вставь свой токен:**
   ```
   BOT_TOKEN=сюда_вставь_токен_от_BotFather
   ```
   
   Пример:
   ```
   BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
   ```

---

### 4️⃣ Установи библиотеки

Открой командную строку **В ПАПКЕ ПРОЕКТА** и выполни:

```bash
pip install -r requirements.txt
```

Это установит все нужные библиотеки (aiogram, aiohttp и др.)

⏱️ Займёт 1-2 минуты

---

### 5️⃣ Запусти бота!

В командной строке выполни:

```bash
python bot/main.py
```

✅ Если всё правильно, увидишь:
```
🚀 Бот запущен!
```

❌ Если ошибка - смотри раздел "Проблемы" ниже

---

### 6️⃣ Протестируй бота

1. Открой Telegram
2. Найди своего бота (по username который создал)
3. Нажми **START** или отправь `/start`
4. Попробуй написать название песни (например: "Наследство")

**Что должно работать:**
- ✅ Команда /start - приветствие
- ✅ Поиск по названию - показывает результаты
- ✅ Кнопки "Избранное", "Рекомендации"
- ⚠️ Скачивание пока НЕ работает (это следующий шаг)

## 🔧 ВОЗМОЖНЫЕ ПРОБЛЕМЫ:

### ❌ "python не является внутренней командой"
**Решение:** Python не установлен или не добавлен в PATH
- Переустанови Python
- При установке поставь галочку "Add Python to PATH"

### ❌ "No module named 'aiogram'"
**Решение:** Не установлены библиотеки
```bash
pip install -r requirements.txt
```

### ❌ "BOT_TOKEN не найден"
**Решение:** Не создан файл `.env` или токен неправильный
- Проверь, что файл называется `.env` (без .txt!)
- Проверь, что токен скопирован полностью
- Не должно быть пробелов: `BOT_TOKEN=токен` (не `BOT_TOKEN = токен`)

### ❌ Бот не отвечает в Telegram
**Решение:**
- Проверь, что бот запущен (в консоли должно быть "Бот запущен!")
- Проверь токен в `.env`
- Попробуй остановить бота (Ctrl+C) и запустить снова
