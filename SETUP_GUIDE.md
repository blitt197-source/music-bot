# 📖 Пошаговая инструкция по запуску бота

## Шаг 1: Установка Python

1. Скачай Python с официального сайта: https://www.python.org/downloads/
2. Выбери версию 3.9 или новее
3. При установке **обязательно** поставь галочку "Add Python to PATH"
4. Проверь установку: открой командную строку и напиши `python --version`

## Шаг 2: Создание Telegram бота

1. Открой Telegram и найди бота **@BotFather**
2. Отправь команду `/newbot`
3. Придумай имя для бота (например: "Мой Музыкальный Бот")
4. Придумай username (должен заканчиваться на "bot", например: "my_music_bot")
5. **Скопируй токен** который даст BotFather (выглядит так: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 3: Настройка проекта

1. Открой папку с проектом в командной строке
2. Создай файл `.env` (скопируй `.env.example` и переименуй)
3. Открой `.env` и вставь свой токен:
   ```
   BOT_TOKEN=твой_токен_от_BotFather
   ```

## Шаг 4: Установка зависимостей

Открой командную строку в папке проекта и выполни:

```bash
pip install -r requirements.txt
```

Это установит все необходимые библиотеки.

## Шаг 5: Запуск бота

В командной строке выполни:

```bash
python bot/main.py
```

Если всё правильно, увидишь сообщение: `🚀 Бот запущен!`

## Шаг 6: Тестирование

1. Открой Telegram
2. Найди своего бота по username
3. Отправь команду `/start`
4. Попробуй найти музыку, написав название трека

## Возможные проблемы

### "python не является внутренней командой"
- Python не установлен или не добавлен в PATH
- Переустанови Python с галочкой "Add to PATH"

### "No module named 'aiogram'"
- Не установлены зависимости
- Выполни: `pip install -r requirements.txt`

### "BOT_TOKEN не найден"
- Не создан файл `.env` или токен не указан
- Проверь, что файл называется именно `.env` (без расширения)
