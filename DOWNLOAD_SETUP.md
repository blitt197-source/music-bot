# 🎵 Настройка скачивания музыки

Я добавил функцию скачивания! Теперь бот может скачивать треки с YouTube.

## 📋 Что нужно сделать:

### 1. Установи FFmpeg

**FFmpeg** нужен для конвертации аудио в MP3.

**Быстрый способ (Windows):**

Открой PowerShell **от имени администратора** и выполни:

```powershell
choco install ffmpeg
```

Если `choco` не установлен, смотри подробную инструкцию в файле `FFMPEG_INSTALL.md`

**Проверка:**
```bash
ffmpeg -version
```

### 2. Обнови библиотеки

```bash
pip install -r requirements.txt
```

Это установит `yt-dlp` для скачивания с YouTube.

### 3. Перезапусти бота

```bash
python bot/main.py
```

---

## 🎯 Как работает скачивание:

1. **Ищешь трек** в Spotify или Яндекс.Музыке
2. **Выбираешь трек** из результатов
3. **Бот автоматически:**
   - Ищет этот трек на YouTube
   - Скачивает аудио
   - Конвертирует в MP3
   - Отправляет тебе

⏱️ Скачивание занимает **10-30 секунд** в зависимости от трека.

---

## ✅ Преимущества:

- 🎵 **Оригинальные треки** (YouTube обычно без цензуры)
- 💾 **Высокое качество** (192 kbps MP3)
- 🚀 **Автоматический поиск** (не нужно искать вручную)
- 🗑️ **Автоочистка** (файлы удаляются после отправки)

---

## ⚠️ Возможные проблемы:

### "FFmpeg не найден"
- FFmpeg не установлен или не в PATH
- Установи по инструкции выше
- Перезапусти командную строку

### "Не удалось скачать трек"
- Трек не найден на YouTube
- Попробуй другой трек
- Проверь интернет-соединение

### Долго скачивает
- Это нормально для первого раза
- YouTube может быть медленным
- Обычно 10-30 секунд
